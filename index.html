Cloud

Was ich bereits über "Cloud" gelernt habe

Definition

Cloud Computing bedeutet die Bereitstellung von IT-Ressourcen (z.B. Server, Speicher, Datenbanken) über das Internet auf Abruf, häufig mit einer nutzungsabhängigen Abrechnung.

Vorteile

  • Skalierbarkeit – einfach hoch- oder herunterskalieren
  • Kostenersparnis – Pay-as-you-go-Modell
  • Hohe Verfügbarkeit und Datensicherung

Typische Dienste

Software as a Service (SaaS), Platform as a Service (PaaS), Infrastructure as a Service (IaaS).
